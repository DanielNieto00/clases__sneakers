# 👟 Tienda de Sneakers - POO

## 📌 Descripción
Este proyecto simula una tienda online de sneakers usando programación orientada a objetos en Python.

## ⚙️ Funcionalidades
- Registro de usuario
- Visualización de productos
- Carrito de compras
- Generación de pedidos
- Pago
- Envío

## ▶️ Cómo ejecutar
```bash
python tienda.py
